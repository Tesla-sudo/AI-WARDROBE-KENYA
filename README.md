# AI-Powered Personal Wardrobe & Outfit Recommendation App (Kenya)

## Overview
MERN-based web app for wardrobe management, AI recommendations, and Kenyan fashion integrations.

## Setup
1. Clone repo: `git clone ...`
2. Install dependencies: `cd backend && yarn install`; repeat for frontend.
3. Env vars: Create `.env` in backend with MONGO_URI, CLOUDINARY_API_KEY, etc.
4. Run: `yarn start` in each folder.
